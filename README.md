# WannaRead
side project1 - book recommand site 'WannaRead' (한글 정식 명칭은 '모읽나')

### [1일차] (06.22)
프로젝트 이름 :   
기존에 국비지원 때 개인 프로젝트로 만들었던 서점 서비스 '모아나' => '모읽나'로 결정!  
영어는 'WannaRead'로 정했다.  

프로젝트 주제 : 도서 추천 사이트  

프로젝트 기능(구현 순서) :   
※ 내가 부족하다고 생각하는 DB 설계 경험, 서버 설정해서 직접 프로젝트를 서비스한 경험을 빠르게 충족시키기 위해서 책장 서비스를 우선 구현 후 AWS에 서버를 띄운다.  

- 책장 서비스
- 로그인 기능
- 책장 공유 서비스
- 크롤링을 대표적으로 온라인서점, 도서리뷰유튜브 몇 개만 크롤링 한다. (추후 더 추가할 예정)  

프로젝트 기간 : 책장 서비스 구현 후 aws 서버 올리는 데까지 일주일, 길게는 2주 생각하고 있다.  
(06.23 ~ 06.30 or 7/7)   

환경 : java, 스프링 프레임워크(필요시 스프링부트), oracle, jpa, aws 등
